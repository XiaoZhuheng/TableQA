# 表格质量分析工具

一个轻量级的表格数据质量检测工具，帮助数据分析师快速识别Excel/CSV文件中的数据质量问题。

## 功能特点

- 🔍 **全面检测**：自动检测10类常见表格质量问题，包括不规范列名、空值、重复数据等
- 📊 **智能评分**：基于严格评分算法，给出0-100分的质量评分和等级评定
- 💡 **优化建议**：针对发现的问题提供具体的改进建议
- 👀 **数据预览**：直观展示数据内容，高亮标注问题单元格
- 📦 **离线可用**：单页HTML文件，无需联网，双击即可运行
- 📁 **批量检测**：支持同时上传多个文件进行快速质量检测
- 🎨 **简洁美观**：现代化界面设计，根据评分动态变色

## 支持格式

- Excel文件：`.xlsx`, `.xls`
- CSV文件：`.csv`

## 使用方法

### 单个文件分析

1. 双击打开 `表格质量分析.html` 文件
2. 选择"单个文件分析"模式
3. 点击"选择文件"或拖拽文件到上传区域
4. 查看分析结果：质量评分、问题列表、优化建议、数据预览

### 批量文件检测

1. 双击打开 `表格质量分析.html` 文件
2. 选择"批量文件检测"模式
3. 点击"选择文件"或拖拽多个文件到上传区域
4. 查看批量检测结果表格和统计摘要

## 质量检测项

| 序号 | 检测项 | 检测内容 | 严重等级 |
|------|--------|----------|----------|
| 1 | 不规范列名 | 检测Unnamed列、空列名 | 严重 |
| 2 | 空行问题 | 检测表格中的空行 | 中等/严重 |
| 3 | 空值过多 | 统计空值占比 | 中等/严重 |
| 4 | 空列问题 | 检测完全为空的列 | 中等 |
| 5 | 表头位置偏移 | 检测表头是否不在第一行 | 中等 |
| 6 | 重复列名 | 检测重复的列标题 | 中等 |
| 7 | 数据类型混合 | 检测同一列中多种数据类型混用 | 轻微 |
| 8 | 重复数据行 | 检测完全重复的行 | 轻微/中等 |
| 9 | 数据结构异常 | 检测只有一列有数据的情况 | 严重 |
| 10 | 行长度不一致 | 检测每行列数不一致 | 轻微 |

## 评分标准

### 评分等级

| 等级 | 分数范围 | 说明 |
|------|----------|------|
| 优秀 | 95-100分 | 表格质量非常好 |
| 良好 | 80-94分 | 表格质量较好，有少量问题 |
| 一般 | 60-79分 | 表格存在一些问题，建议优化 |
| 较差 | 40-59分 | 表格存在较多问题，需要优化 |
| 严重 | 0-39分 | 表格质量很差，必须优化 |
